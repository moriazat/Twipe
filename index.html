Twipe

Twipe is a fun application to create a special form of ASCII-Art. It takes an image and converts it to another image which is visually similar to the black-and-white version of the original image.

It is very straight forward and here is how to do it:

Step 1: Click on "Browse" and choose the image you want to convert.

Step 2: Choose the settings you prefer.

Choose the font set.

From the first combo box you can choose the font collection set that you want to be used in the process. If you choose "Single Font", you can choose the specific font family from the dropdown box below.

Check "Use transparent tiles", if you want the characters to write on a transparent canvas.

Choose the tile size.

Note: The tile size affects the output image size. For example, if you give an image of size 200x300 pixels and set the tile size to 20, the output image size would be 4000x6000 pixels. Obviously, this also affect the duration of the whole process.

Step 3: Click on "Convert" button.

Now just wait for the process to finish, and you will get the result next to the original image.
