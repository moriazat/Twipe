Twipe

Twipe is a fun application to create a special form of ASCII-Art. It takes an image and converts it to another image which is visually similar to the black-and-white version of the original image.

Here is how to do it:

Step 1: Open an image in the application.

Step 2: Choose the settings you like.

Choose the font set.

Choose if you want the characters to write on a transparent canvas.

Choose the tile size.

Note: The tile size affects the output image size. For example, if you give an image of size 200x300 pixels and set the tile size to 20, the output image size would be 4000x6000 pixels. Obviously, this also affect the duration of the whole process.

Step 3: Click on Convert button.
